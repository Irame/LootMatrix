# LootMatrix

This is a addon for World of Warcraft to help you find which mythic+ dungeon or raid boss drops loot that you need.

## How to use

To use just type `/lm` or use the addon compartment. The addon lists all Items dropped for a specific class or spec from all dungeons/bosses in a matrix where the rows are dungeons/bosses and columns are slots.

With the search options you can look for items that have the secondary stats you need.
